Hard Life
A girl is fighting for her place in a hostile world, against overwhelming odds. Without help, she'll be lost. I made this as a "quickshot" adventure, set in the world of the "Blue Planet" game by Biohazard. Character sheets have been provided for the first edition of the game, however "Blue Planet" has since been republished by Fantasy Flight games with new rules. The scenario contains hints at child abuse, use this at your own discretion.

gameXF - a roleplaying XML format
This was an attempt to create a generic XML format to store roleplaying information. The package contains an XML schema, XSL Transformation stylesheets and sample data (character descriptions from a "Feng Shui" and an "Unknown Armies" campaign). I doubt that this project will go any further, but you may use the ideas if you like. You'll need an XSLT engine to do the stylesheet transformations.
