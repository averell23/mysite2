Problems? Probleme?

A messed page layout may be caused by lacking CSS (Cascading Style Sheet) support in your browser. The 4- and early versions of the popular browsers (Netscape and Internet Explorer) support CSS only partly, if at all. The recent versions of Netscape, Mozilla and Internet Explorer have sufficient CSS support (I recommend Mozilla). If you have an older PC have a look at Opera, a lightweight browser with very good standards support.

You may also notice that some images are slightly odd in the Internet Explorer (intransparent background). This is a known problem of the IE, which does not handle transparent PNG images correctly. This may be fixed in the next Windows/IE update (in 2004). If you want a more standards-compliant browser, use Mozilla.

Technicalese

These pages aren't optimised for anything. To you, it should be pure HTML 4.01 and CSS 2.0. This still means you'll need a current (i.e. CSS-compliant) browser. like Mozilla or Opera. The Internet Explorer will also do, although it still has difficulties coping with the standards (and the Intel x86 version still doesn't handle PNG transparency properly - the Mac version is reported to do fine). If you have trouble viewing the pages, try one of the browsers mentioned above.

If you use an older or otherwise non-compliant browser (like the Netscape Navigator 4.x) you should still be able to view these pages, but you'll loose the original layout. Text-only browsers should do just fine.
